# Tool
## HTTP接口测试工具 V0.1.0
目前功能：
1. 发送（get/post）请求
2. 添加删除请求参数
3. 支持跨域请求

页面需要开启web服务器运行，建议使用php自带服务器。
命令如下：
> php -S 0.0.0.0:8001

然后再游览器中输入地址：http://127.0.0.1:8001/test_tool/time.html

即可打开页面

> 建议打开控制台console查看返回的格式化数据
建议打开控制台network查看http请求的hearders信息
